ITV Usage Guide

Thanks for installing ITV!

Thanks for installing the "In-Tab Values" chrome extension. Listed here are usage instructions and activation details to help you get the most out of ITV's numerous features.

Activation *IMPORTANT*

Before starting use of ITV, you will need to purchase a VIP Server Subscription from our hub game below:


https://www.roblox.com/games/6433970189

Once you have acquired a subscription, ITV will be automatically enabled, and will be active on your trading pages.

Features

You can enable or disable any of these features from the options menu, which can be accessed by clicking the extension icon , which can be found in your extensions toolbar.

  • Enabled - Activates the extension / see Rolimons item values and calculated totals displayed within your trading tabs!
  • Overrap/Underrap data - Get indicators of whether items are likely to raise/lower in value depending on their RAP.
  • Side Value Preview - See values directly embedded onto the trades list. To combat Roblox's API limits, we are forced to use caching, so you may have to wait a few minutes before all of your trades are affected when first using this feature. This currently works for inbounds only, and if you have more than 100 trades inbound, only the first 100 will be loaded. If you have this option enabled, you will also see a "Cache Status" above your trade list, which tells you if all of your trades have been cached and will appear in "Side Value Preview".
  • Inbound Trade Filterer - When enabled, you will be shown a GUI on your trades page. It can be used to automatically decline inbound trades, that are not profitable. If you have more than 100 inbounds, then it will only take into account the first 100 before needing to cache again. Options:
    • Projected - When enabled, this will tell the program to decline trades where you are offered a projected item.
    • Values - Enabling this will mean that the program will calculate trade values, and decline ones that don't correspond with the "Decline Strictness".
    • Decline Strictness - *Requires "Values" to be enabled* This setting will determine which trades to keep, based off a filter option you set. The different options are:

    - all losses: Will decline all trades where you lose value
    - all trades: Will decline all trades you have inbound
    - small losses: Will decline trades where you lose more than 5% of the your offer value
    - medium losses: Will decline trades where you lose more than 10% of the your offer value
    - large losses: Will decline trades where you lose more than 30% of the your offer value

    • Start - Will begin the scanning process, and automatically decline inbound trades based off the parameters you set. If you get the error "Caching in progress. Please try again later.", it means that the caching system has not yet discovered all of your trades, and you should wait a few minutes before trying again.
  • User Profile Value - View user's total value from their profiles.
  • Total Win/Loss - Shows a positive or negative number to the right of the "Decline" button to represent how much of a win/loss the current trade is.
  • Flagging - Be notified of item properties, such as rarity, projection, or being proof based.
  • MRGA for Rares - Use Rarenthusiast values for rare items, instead of Rolimons( or instead of rblx.trade if you have "rblx.trade values instead of Rolimon's" enabled ).
  • rblx.trade values instead of Rolimon's - Use rblx.trade values instead of Rolimons.
  • Item RAP Chart Click the graph icon to see an item's RAP graph embedded at the bottom of the page.
  • Hide Serials, Usernames, Dates Censors according data on your trades page so you don't leak information while sending trade screenshots.
  • Quick Links Adds links to pages on rolimons.com and rblx.trade such as:
    A link to user profiles next to usernames in a trade,
    a link to each item's general page next to the item's value,
    and a link to each item's specific UAID page accessed by clicking the item's serial button

    • You can view this page at any time by clicking "Usage Guide" from the options menu.
